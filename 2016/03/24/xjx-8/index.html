关于递归回溯的问题总结（一）

发表于 2016-03-24

最近刷leetcode遇到了很多递归回溯的问题，在这里总结下

问题一:Subsets

Given a set of distinct integers nums, return all possible subsets

数学中的全排列问题，唯一需要注意的是，所有的排列都是非递减的，所以在求全排列之前先排序

问题一的进阶 Subsets II

这一次在输入当中会出现重复数字，例如[1,2,2]
输出：
[
[2],
[1],
[1,2,2],
[2,2],
[1,2],
[]
]

#问题三 Combinations

If n = 4 and k = 2, a solution is:

[
[2,4],
[3,4],
[2,3],
[1,2],
[1,3],
[1,4],
]
